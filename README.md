# House Price Prediction Using Machine Learning

## Overview

This is a beginner Machine Learning project that predicts house prices using the California Housing dataset. The project uses a Random Forest Regression model and Scikit-learn for data preprocessing and prediction.

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Joblib

## Project Files

```
housing.csv      # Dataset
input.csv        # Input data
output.csv       # Predicted results
main.py      # Main Python script
```


## Output

The predicted house prices are saved in `output.csv`.
